First, there is the information at the link https://opensuse.github.io/openSUSE-docs-revamped-temp/best_of_post/ 

1. Get wifi working
2. Run sudo zypper ref and sudo zypper dup. Reboot. Now you're updated.
3. Change hostname (sudo hostnamectl set-hostname NEWHOSTNAME). Check this with command hostname.
4. Install git and stow
5. Install ntfs-3g
6. Set the time if necessary.
7. Install Git and clone your computer setup repo
8. Get your data drive mounted (see add-data-drive.txt) 
9. Download and install packages on Zypper
10. Install opi by running sudo zypper in opi
11. Install other programs as necessary (from OPI, flathub, etc.)
12. Clone dotfiles repo and set up dotfiles
13. Move other files from backup as necessary
14. Move rclone config file from backup
15. Move scripts into .local/bin and make them executable with sudo chmod u+x *
16. Start configuring DE or window manager
17. Remove and lock packages as needed
18. Download printer drivers and set up printer. Add “ipp” and “mdns” services to Public zone in firewall so that wireless printing can work. See add-printer.txt
19. Install Microsoft fonts by installing Zypper package fetchmsttfonts
